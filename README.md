## BemibbsPHP Framework 1.0
安装:
```
composer create-project bemiou_lingqi/bemibbs bemi
```
## 功能特性
+ 插件引擎
+ 模板引擎
+ 路由功能
+ 自定义配置
>需要PHP版本7.0+
